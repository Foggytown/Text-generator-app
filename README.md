Небольшое приложение сделанное с помощью пакета Reflex на python в рамках домашнего задания https://github.com/ashaba1in/hse-nlp/blob/2025/homeworks/hw1_n_gram_generaiton.ipynb. Изначально перед пользователем предстает страница загрузки модели которая происходит асинхронно. После загрузки модели пользователь нажимает на кнопку и переходит на страницу непосредственного взаимодействия с генератором текста. При наборе текста модель предлагает до пяти разных вариантов окончания слова, каждый из которых можно выбрать нажам на него мышкой или выбрав стрелочками вверх вниз и нажав Enter, при нажатии Esc подсказки исчезают. При наведении на подсказку также в окне наборе серым текстом будет написано выбранное продолжение слова а также два слова после этого которые предсказала модель. Дополнительно справа сделана кнопка удалить все для удобства.
